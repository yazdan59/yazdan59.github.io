01 — Overview

What is GCPBayes
Pipeline?

Pleiotropy — where a single genetic locus influences multiple phenotypes — is central to understanding the shared genetic basis of complex diseases. The GCPBayes Pipeline provides a rigorous, end-to-end approach to detecting such shared loci at the gene level using GWAS summary statistics.

Unlike SNP-level approaches, gene-level Bayesian inference aggregates weak signals across all SNPs in a gene, increasing statistical power while retaining biological interpretability. The pipeline was applied to study shared genetic architecture between Breast Cancer (BCAC 2020) and Ovarian Cancer (OCAC).

PLACO — PLeiotropic Analysis under COmposite null hypothesis
SNP-level pre-screening; identifies genome-wide significant pleiotropic variants (p < 5×10⁻⁸)
GCPBayes DS() — Dirichlet-Spike model
Gene-level Bayesian MCMC; estimates θ (pleiotropic probability) and PPAs per study
Annotation — GENCODE v38 lifted to hg19
Maps SNPs to coding genes; adds gene-level metadata (type, length, SNP density)
✦ Recommended
With LD Clumping
Pre-processes PLACO output with PLINK LD clumping (r²=0.8) to retain only independent SNPs per gene window. Reduces collinearity and computation time.

Alternative
Without LD Clumping
Retains all annotated SNPs within gene boundaries. No PLINK dependency. May capture more signal at the cost of collinearity and longer runtime.

02 — Workflow

Step-by-Step
Pipeline

Run scripts in order. Edit the DEFINITION SECTION at the top of each script to set your local file paths.

0
GWAS Reformatting
⏱ ~3 min each
0_code_reformatting_file_bcac_2020_all.py  /  0_code_reformatting_file_ocac_bcac_2020_all.py
Python scripts that parse raw GWAS summary statistics from BCAC 2020 and OCAC into a standardized tab-separated format. Dataset-specific; must be adapted for other GWAS inputs.
▼ Input
icogs_onco_gwas_meta_overall_...
extraction_OCAC.txt
▲ Output
BCAC_2020_onco_ALL_reformatted.txt
OCAC_BCAC_2020_onco_ALL_reformatted.txt
1
Common SNP Extraction + Z-scores
⏱ ~3 min
1_code_find_common_snps_one_pair.R
Filters both GWAS files (info > 0.8, MAF > 0.01), computes Z = β/SE (sentinel value 1×10⁸ when SE = 0), finds the SNP intersection via inner join on (snp, chr, bp_hg19), removes duplicates, sorts by SNP ID. Output files are direct PLACO input.
▼ Input
BCAC_2020_onco_ALL_reformatted.txt
OCAC_BCAC_2020_onco_ALL_reformatted.txt
▲ Output
BCAC_2020_ALL_Common_OCAC_inc_Z.txt
OCAC_Common_BCAC_2020_ALL_inc_Z.txt
2
Pleiotropy Screening — PLACO
⏱ ~3.5 hours
2_code_run_PLACO_decor_one_pair.R
Builds Z-matrix and P-matrix. Tests Pearson correlation between trait Z-scores; if p < 0.05, decorrelates via R^(−½) transformation. Estimates variance parameters (var.placo), runs placo() for each SNP, applies FDR correction. Significant threshold: p.placo < 5×10⁻⁸. Sign of T.placo distinguishes same-direction (+) from opposite-direction (−) effects.
▼ Input
BCAC_2020_ALL_Common_OCAC_inc_Z.txt
OCAC_Common_BCAC_2020_ALL_inc_Z.txt
▲ Output
output_PLACO_BCAC_2020_ALL_OCAC.txt
(T.placo, p.placo, FDR.placo per SNP)
3
LD Clumping via PLINK
⏱ ~7 hours
3_code_ldclumping_08_local.R   (skip for no-clumping workflow)
Runs PLINK clumping chromosome by chromosome (chr 1–22). Parameters: clump_kb=10000, clump_r2=0.8, clump_p=0.99. Uses 1000 Genomes EUR reference panel. Requires PLINK binary via genetics.binaRies::get_plink_binary(). At r²=0.8, highly correlated SNPs are removed while haplotype block representatives are retained.
▼ Input
output_PLACO_BCAC_2020_ALL_OCAC.txt
1000G EUR reference (bed/bim/fam)
▲ Output
output_ld_clumping_08_BCAC_2020_ALL_OCAC.txt
4
GCPBayes Input Preparation (5-Step Pipeline)
⏱ ~6 min
4_code_pipeline_annot_coding_ldclumping_extra_info.R
Internal 5-step pipeline: (1) Re-filter GWAS (info>0.9, MAF>0.05), find common SNPs. (2) Join with SNP-to-gene annotation (ANNOT ≠ "."). (3) Merge both GWAS traits with gene labels. (4) Intersect with LD output to retain independent SNPs. (5) Clean gene labels (split multi-gene SNPs by "|", remove variant annotations), build Betah/Sigmah list structures per gene, build extra_info list with genomic metadata. Sorted by ascending SNP count.
▼ Input
GWAS files (both traits)
Annot_BCAC_coding.txt
output_ld_clumping_08_...txt
annot_gencode_v38lift37_...txt
▲ Output
Matrices_output_pipeline_...Rdata
Matrices_extra_info_...Rdata
5
GCPBayes Run — Genes ≤ 500 SNPs
⏱ ~1.5 days
5_code_gcpbayes_less_extra_info.R
Runs DS() (Dirichlet-Spike model) for each gene. MCMC: niter=3000, burnin=1000, nthin=2, nchains=1. Priors: κ₀=0.5, σ²₀=1, a₁=a₂=d₁=d₂=0.1. Genes with θ > 0.5 are flagged pleiotropic. Two SNP identification criteria: credible interval (CI) and posterior median thresholding. Errors caught via try(); logged to errors file.
▼ Input
Matrices_output_pipeline_...Rdata
Matrices_extra_info_...Rdata
▲ Output
..._less_threshold_500_results.txt
..._less_threshold_500_pleiotropy.txt
..._less_threshold_500_errors.txt
6
GCPBayes Run — Genes > 500 SNPs
⏱ up to ~1 week
6_code_gcpbayes_greater_extra_info.R
Identical to Step 5 but processes the computationally intensive genes with > 500 SNPs. Separated to allow independent HPC job submission or parallel execution. Same GCPBayes DS() parameters apply. Steps 5 and 6 can run concurrently.
▼ Input
Matrices_output_pipeline_...Rdata
Matrices_extra_info_...Rdata
▲ Output
..._greater_threshold_500_results.txt
..._greater_threshold_500_pleiotropy.txt
..._greater_threshold_500_errors.txt
Analysis scripts (post-run):
code_analysis_PLACO_results_one_pair.R — Manhattan plot (ggplot2), Pearson/Spearman Z-score correlation, significant SNP export.
code_analysis_gcpbayes_input_ggplot.R — SNP distribution histogram and bar chart from the Rdata input file.

04 — Outputs

Output File
Schema

GCPBayes produces three file types per run. The pleiotropy file is the primary result of interest.

▸ Pleiotropy Output File (candidate pleiotropic genes — θ > 0.5)

Column Description
GeneNameHGNC gene symbol
#SNPs / ThetaNumber of SNPs in gene after pipeline filtering; posterior pleiotropic probability (0–1)
Pleiotropic_Effect_based_on_CIYes/No — pleiotropic SNPs identified via credible interval criterion
Pleiotropic_Effect_based_on_MedianYes/No — pleiotropic SNPs identified via posterior median thresholding
chr / start / endGenomic coordinates in hg19
gene_typeGENCODE biotype (e.g., protein_coding)
gene_lengthGene span in base pairs (end − start)
snp_number_gene_length_ratioSNP density (SNPs per bp)
SNP_IDIdentifiers of pleiotropic SNPs (space-separated)
Study1 / Study2 / TotalPer-SNP significance indicators per trait and combined

▸ Full Results File (all tested genes)

ColumnDescription
gene_index / geneInternal gene index; HGNC gene name
log10BFlog₁₀ Bayes Factor — evidence for pleiotropy vs. null
lBFDRLocal Bayesian FDR — lower = more confident
thetaPosterior probability of pleiotropic effect (threshold: 0.5)
PPA1 / PPA2Posterior Probability of Association per study (Trait 1 / Trait 2)
SNP_numbers / run_timeSNPs analyzed; computation time in seconds
Interpretation: High θ (near 1) + high log10BF + low lBFDR = strong pleiotropic evidence. The two SNP-level criteria (CI vs. median) may differ slightly; their intersection is the most conservative and robust set.
05 — Visualization

Shiny App

An interactive R Shiny application for visualizing GCPBayes pleiotropy output. Upload your results file and explore candidate genes instantly.

Live app: https://cespexpher.shinyapps.io/gcpbayesoutput/
Two versions: with karyotype (requires biomaRt/karyoploteR, slower) and without karyotype (faster, lighter dependencies). Input: CSV or tab-separated GCPBayes pleiotropy output file, max 30 MB.
📂
Upload File
File upload with configurable separator, quote character, and header detection. Preview data table.
⬤
Scatter Plots
Dynamic X/Y axis selection from numeric columns. Download as PDF.
📊
Theta Histogram
Dual histograms: zoomed (θ 0.5–1, binwidth=0.02) and full range (binwidth=0.05). Count labels on bars.
🔵
Circos Plot
BioCircos genomic circular plot: inner ring = gene length, outer ring = SNP count, color-coded per chromosome.
🥧
Pie Chart
Interactive Plotly pie showing distribution of candidate pleiotropic genes per chromosome.
📦
Box Plot
Three-panel gene length distribution: boxplot, jittered boxplot, log₁₀-scale violin plot.
🧬
Karyotype (chr)
Per-chromosome view with gene markers labeled via biomaRt. Select chromosome from dropdown. (With-karyotype version only.)
🗺️
Karyotype (All)
Full genome karyotype (chr1–22) with all candidate pleiotropic genes labeled. Download as PDF.
